### 问题
- 飞书渠道每次收到消息后返回错误：`<400> InternalError.Algo.InvalidParameter: Range of input length should be [1, 196608]`

### 解决方案
- 清理飞书去重缓存文件：`~/.openclaw/feishu/dedup/default.json`
- 重启 OpenClaw Gateway
