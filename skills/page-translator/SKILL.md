# 网页翻译技能

## 使用场景

当用户需要翻译当前打开的网页时使用。

## 操作步骤

1. **获取网页内容**
   - 使用 `browser` 工具的 `snapshot` 或 `screenshot` 获取当前页面
   - 如果需要更精确的翻译，使用 `web_fetch` 获取页面文本内容

2. **翻译内容**
   - 将获取到的页面内容发送给大模型
   - 使用提示词："请将以下网页内容翻译成中文，保持原文格式和结构"

3. **返回结果**
   - 将翻译后的中文内容展示给用户
   - 如果页面很长，可以先翻译主要标题和内容摘要

## 示例触发语

- "翻译这个页面"
- "翻译此页面"
- "翻译网页"
- "帮我翻译"
- "页面是英文的"
- "翻译一下"
